# SPLIT_RGBir_jpg
script per splittare immagini jpg (rgb + ir) con banda infrarosso - bash shell di unix

## come fare con installazione QGIS da OSGeo4W

1. avviare `MSYS`;
2. navigare fino alla cartella che contiene le immagini; (se la cartella (rgb che contiene le immagini) Ã¨ sul desktop: cd C:/Users/utente/Desktop/rgb)
3. copiare ed incollare lo `script` nella `shell`;
4. invio
